5.18.5. Status codes and symbols — statno¶

The module makes all the status symbols available that are used in libxdrfile2 (and which are described there). The value of each symbol is an integer (as defined in xdrfile.h).

The dictionary xdrfile.errno.errorcode maps numeric codes to symbol names.
