5.3. DL Poly format Topology Readers — MDAnalysis.topology.DLPolyParser

Read DL Poly format topology files

DLPoly files have the following Attributes:

  • Atomnames

  • Atomids

Guesses the following attributes:

  • Atomtypes

  • Masses
